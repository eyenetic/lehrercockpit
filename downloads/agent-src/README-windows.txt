Cockpit Agent fuer Windows
==========================

1. ZIP entpacken
2. Outlook oeffnen
3. Doppelklick auf "cockpit-agent.bat"
4. Lehrercockpit neu laden

Der Agent startet lokal auf:
http://127.0.0.1:8765

Er liest nur:
- Absender
- Betreff
- Datum
- Gelesen/Ungelesen

Mailinhalte werden nicht gelesen.
