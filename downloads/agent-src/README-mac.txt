Cockpit Agent fuer Mac
======================

1. ZIP entpacken
2. Doppelklick auf "cockpit-agent.command"
3. macOS nach Mail-Zugriff fragen lassen und bestaetigen
4. Lehrercockpit neu laden

Der Agent startet lokal auf:
http://127.0.0.1:8765

Er liest nur:
- Absender
- Betreff
- Datum
- Gelesen/Ungelesen

Mailinhalte werden nicht gelesen.
